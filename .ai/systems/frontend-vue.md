# Frontend Vue

The active frontend is under `frontend/`.

## Stack

- Vue 3
- TypeScript
- Vite
- Pinia
- vue-router
- pnpm

## Commands

```powershell
npx pnpm --dir frontend run typecheck
npx pnpm --dir frontend run lint:check
npx pnpm --dir frontend run build
```

## Notes

- Do not use old `web/default` React/Bun instructions from `F:\newAPI`.
- Keep Fuxi API branding and production Chinese usage in mind.
- Avoid changing payment, OAuth, or legal text without checking runtime settings and deployment implications.
